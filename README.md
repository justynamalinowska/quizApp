# Quiz App

## Rozpoczęcie pracy

1. **Zainstaluj zależności:**
   ```sh
   npm install
2. **Uruchom serwer deweloperski:**
   npm run dev
Otwórz http://localhost:3000 w przeglądarce.
3. **Zbuduj wersję produkcyjną:**
npm run build
npm run start
Uruchamianie testów
1. **Zainstaluj Playwright:**
npm install @playwright/test --save-dev
2. **Uruchom testy:**
npx playwright test
3. **Wyświetl raport z testów:**
npx playwright show-report

Struktura projektu
app/: Kod źródłowy aplikacji.
tests/: Pliki testowe Playwright.
playwright.config.js: Plik konfiguracyjny Playwright.

Zależności:
-Next.js
-React
-Firebase
